# 📦 Demand Forecasting using Machine Learning

A complete **end-to-end Data Science project** that predicts product demand using machine learning techniques, incorporating pricing, promotional, temporal, and external factors.

---

## 🚀 Overview

This project focuses on building a **robust demand forecasting system** to help businesses optimize inventory and sales strategies.  
It leverages real-world features such as pricing, competition, seasonality, weather conditions, and promotions to improve prediction accuracy.

---

## 🧠 Key Features

- 📊 Data preprocessing and feature engineering  
- 🤖 Multiple model training and comparison  
- ⚙️ Hyperparameter tuning using RandomizedSearchCV  
- 📈 Model evaluation using R² and MAE  
- 🔍 Feature importance analysis for interpretability  
- 🌐 Interactive Streamlit web application for real-time predictions  

---

## 🛠️ Tech Stack

- **Languages & Libraries:** Python, Pandas, NumPy  
- **Machine Learning:** Scikit-learn, XGBoost  
- **Visualization:** Matplotlib  
- **Deployment:** Streamlit  

---

## ⚙️ Machine Learning Workflow

### 1. Data Preprocessing
- Handling categorical variables using encoding  
- Extracting temporal features (Month, Weekday)  
- Ensuring no data leakage  

### 2. Model Building
- Linear Regression (baseline)  
- Random Forest  
- XGBoost (final model)  

### 3. Model Optimization
- Hyperparameter tuning using RandomizedSearchCV  
- Cross-validation for robustness  

### 4. Evaluation
- Metrics: R² Score, Mean Absolute Error (MAE)  

---

## 📊 Key Insights

- Epidemic and promotional activities significantly influence demand  
- Pricing has lower impact compared to marketing strategies  
- External factors like weather and region affect buying behavior  
- Seasonal trends exist but are secondary drivers
